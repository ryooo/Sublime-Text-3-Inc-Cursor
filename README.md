#Inc-Cursor
## Overview
Increase multi cursor values.
Initial value is the first cursor value.
Cycled enumerations on the configured value support.

## Uses

## Default Settings
inc_cursor.sublime-settings
```json
{
  "enums": [
    ["日", "月", "火", "水", "木", "金", "土"],
    ["Sun", "Mon", "Tue", "Wed", "Thu", "Fri", "Sat"]
  ]
}
```
